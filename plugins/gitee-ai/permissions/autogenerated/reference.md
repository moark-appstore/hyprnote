## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-send-verification-code`
- `allow-verify-code`
- `allow-get-user-info`
- `allow-pay`
- `allow-get-pay-result`
- `allow-get-app-info`
- `allow-save-token`
- `allow-get-login-status`
- `allow-logout`
